Namespaces

  • Problem : You cannot have tow classes with the same name
  • Namespaces allow you to have multiple classes with the same name.
  • Namespaces can prevent name collision
  • Namespaces can make our code more organized

Namespaces

  • Any valid php code can be in a namespace
  • Only classes, constants, functions, and interfaces are affected
